Smart Campus Demo - Initial Deployment (Version 1.0)

A single-file CI/CD demo: push & watch it update on Azure Static Web Apps.

What this demo shows

  • GitHub Actions triggered on push → GitHub Actions pipeline
  • Automated deployment to Azure Static Web Apps
  • Immediate visible change on the live URL when index.html is updated

Quick edit steps

  1. Edit the <h1> line and change the version (e.g., 1.0 → 1.1).
  2. git add . && git commit -m "Bump to v1.1" && git push
  3. Watch GitHub Actions and refresh the live site.
Deployed via: GitHub Actions → Azure SWA
